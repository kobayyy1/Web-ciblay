### 📌 Deskripsi Proyek
Sistem Informasi Terpusat **Lembaga Penelitian dan Pengabdian kepada Masyarakat (LPPM)** Institut Seni Indonesia Padangpanjang. Platform ini menjadi gerbang utama dalam mendokumentasikan riset ilmiah, publikasi berita, dokumentasi media, serta papan informasi kelembagaan yang transparan.

### 🚀 Fitur Utama
* **🌍 Multi-Language Sync:** Integrasi translasi real-time yang tersinkronisasi di seluruh halaman menggunakan LocalStorage.
* **📊 Admin Dashboard:** Monitoring data penelitian, berita, media, dan informasi secara real-time.
* **📂 Manajamen Data:** CRUD lengkap untuk data profil pimpinan, penelitian, hingga dokumentasi media.
* **✨ Interaktif Slider:** Implementasi *Visual Distance Detector* pada carousel informasi untuk pengalaman navigasi yang presisi.
